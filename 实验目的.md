# 实验目的

## 一、总体目标

本实验以「脊椎CT椎体逐节分割」为应用场景，构建一套完整的医学图像处理系统。核心目标是将课程前13个实验所学的全部图像处理技术串联起来，形成一条从原始医学影像到最终AI分割结果的完整数据处理流水线，最终基于深度学习（U-Net）实现椎体的自动逐节分割与量化评估。

## 二、具体目标

### 2.1 医学影像基础能力（实验1、12）
- 掌握 NIfTI 格式 3D 医学影像的读取、元数据解析与存储
- 理解 CT 值的物理含义（Hounsfield Unit），掌握窗宽窗位调节原理（骨窗/软组织窗/肺窗），能根据图像直方图自动计算最优窗宽窗位
- 掌握 MPR 多平面重建技术，从 3D 体数据中提取轴状面、冠状面、矢状面正交切片，理解三维医学影像的空间关系

### 2.2 图像预处理技术链（实验2、3、4、5、11）
- **灰度变换**：手动实现线性变换、对数变换、Gamma 幂次变换、分段线性变换，理解不同变换函数对图像对比度和亮度的调节规律
- **直方图处理**：手动实现灰度直方图统计、计算均值/方差/偏度/峰度/能量/熵等统计特征，手动实现直方图均衡化以增强图像对比度
- **插值与重采样**：手动实现最近邻插值和双线性插值，理解不同插值核对图像缩放质量的影响；实现灰度级量化，观察量化级数对图像质量的影响
- **频域滤波**：手动实现二维 FFT/IFFT（Cooley-Tukey 基-2 递推算法），理解频谱中心化的意义；手动实现理想/巴特沃斯/高斯三种低通和高通滤波器，理解频域滤波在图像平滑和边缘增强中的应用
- **色彩空间与伪彩色**：手动实现 RGB↔HSI 色彩空间转换，理解色调、饱和度、亮度的分离；实现多种伪彩色映射方案（jet/hot/bone），增强人眼对灰度差异的感知

### 2.3 传统图像分割技术链（实验6、7、8、9）
- **边缘检测**：手动实现 Roberts、Sobel、Prewitt 一阶梯度算子以及 Canny 完整四步流程（高斯平滑→梯度计算→非极大值抑制→双阈值边缘连接）；手动实现 Laplacian 二阶微分算子和 LoG（高斯拉普拉斯）算子，理解一阶与二阶边缘检测的差异
- **形状与结构分析**：手动实现霍夫直线检测，将图像空间的直线映射到参数空间的峰值检测，应用于椎体终板边缘识别；实现傅里叶描绘子，对分割轮廓进行频域形状描述与重建
- **纹理特征提取**：手动实现四方向（0°/45°/90°/135°）灰度共生矩阵（GLCM），提取对比度、能量、熵、逆差分矩、自相关五种 Haralick 纹理特征；手动实现 LBP 局部二值模式和 Gabor 小波多尺度纹理滤波
- **阈值分割**：手动实现迭代阈值法和 Otsu 最大类间方差法，理解自动阈值选取的数学原理
- **区域与聚类分割**：手动实现区域生长算法（BFS 队列搜索、灰度差阈值准则、八邻域生长），支持交互式种子点选取；手动实现 K-means 聚类分割（灰度特征 + 空间坐标特征两种方案）

### 2.4 图像配准与量化评估（实验10）
- **图像配准**：手动实现 2D 刚性配准（旋转+平移）和仿射配准（缩放+旋转+剪切+平移），使用 MI/NMI/MSE 作为相似性度量，采用梯度下降优化变换参数
- **量化评估指标**：手动实现 Dice 系数、IoU（交并比）、MSE（均方误差）、MI（互信息）、NMI（归一化互信息）、PSNR（峰值信噪比），建立客观的分割精度评价体系
- **形态学后处理**：手动实现二值腐蚀、膨胀、开运算、闭运算，用于分割结果的后处理去噪与孔洞填充
- **综合对比**：实现自动化的多方法对比评估器，对 Otsu/迭代阈值/区域生长/K-means/深度学习五种方法生成统一的 Dice/IoU/MSE/MI/NMI 指标对比报告

### 2.5 深度学习方法（综合应用）
- **网络结构**：实现经典 2D U-Net（编码器-解码器 + 跳跃连接），理解卷积、池化、上采样、特征拼接在医学图像分割中的作用
- **数据工程**：从 3D CT 体数据和标注中构建 2D 切片训练集，实现数据增强流水线（灰度扰动、随机翻转），处理类别标签重映射问题
- **训练与推理**：实现完整的训练循环（Dice Loss + Cross Entropy 联合损失），支持 GPU 加速推理；将深度学习分割结果与传统方法进行定量对比
- **后处理**：结合连通域分析和椎体空间位置先验，对分割结果进行去噪、编号

### 2.6 系统集成（实验13）
- 基于 PyQt5 构建图形化交互界面，将上述所有功能模块集成到统一平台中
- 实现 MPR 三视图实时显示、原图/结果双画布对照、种子点鼠标交互选取、参数滑块实时调节等功能
- 掌握 GUI 与后台计算线程的协作，避免长时间计算阻塞界面响应

## 三、技术路线

```
原始CT体数据 (NIfTI, 161x338x61)
        │
        ├── 窗宽窗位调节、MPR三平面重建 ────→ 矢状面切片 (161x338 px)
        │
        ├── 预处理 ────────────────────────→ 灰度变换、直方图均衡化、频域滤波、伪彩色
        │
        ├── 传统分割基线 ───────────────────→ Otsu、区域生长、K-means
        │                                      边缘检测、纹理分析
        │
        ├── 深度学习分割 ───────────────────→ 2D U-Net 训练/推理/后处理
        │
        └── 量化评估 ───────────────────────→ Dice、IoU、MI/NMI
                                               四种方法对比报告
```

通过本实验，将孤立的知识点串联为一条完整的数据处理链路——从原始医学影像的读取与可视化，经过多种预处理和传统分割方法的尝试，最终利用深度学习实现高质量的自动化椎体分割，并通过量化指标客观评估各类方法的性能优劣。
