### 3.2 🔧 预处理 — 覆盖实验 2、3、4、5、11

#### 3.2.1 窗宽窗位调节（实验 12）

| 预设 | 窗宽/窗位 | 用途 |
|------|-----------|------|
| **骨窗** | WW=1500, WL=300 | 观察骨结构（椎体最佳） |
| **软组织窗** | WW=400, WL=40 | 观察肌肉、椎间盘 |
| **肺窗** | WW=1500, WL=-500 | 观察肺组织 |
| **脑窗** | WW=80, WL=40 | 观察脑组织 |
| **纵隔窗** | WW=400, WL=40 | 观察纵隔结构 |

> **操作**：下拉选预设 → 点「应用窗宽窗位」→ MPR三视图实时刷新

#### 3.2.2 灰度变换（实验 3）

| 变换类型 | 参数 | 效果 |
|----------|------|------|
| **线性变换** | 斜率=亮度增益 | s = a·r + b |
| **对数变换** | c=尺度 | 增强暗部，压缩亮部 |
| **Gamma变换** | γ<1 增亮暗部 | s = c·r^γ |
| **分段线性** | 自动 | 对比度拉伸 |

> **操作**：选变换类型 → 调 Gamma/斜率滑块 → 点「应用灰度变换」→ 右画布看结果

#### 3.2.3 直方图均衡化（实验 4）

点「**直方图均衡化**」→ 灰度分布拉平，对比度增强。左边原图，右边结果，可直接对比。

#### 3.2.4 频域滤波（实验 5）

| 滤波器 | 说明 |
|--------|------|
| **高斯低通** | 平滑/去噪，保留大结构 |
| **高斯高通** | 边缘增强，突出细节 |
| **理想低通** | 锐截止，可能有振铃 |
| **巴特沃斯低通** | 折中方案，无振铃 |

> **操作**：选滤波器 → 设截止频率(1-100) → 点「应用频域滤波」

#### 3.2.5 重采样与量化（实验 2）

- **重采样**：选最近邻/双线性 → 设倍率(0.1-4.0) → 点「重采样」
- **灰度量化**：设灰度级(2-256) → 点「灰度量化」查看量化效果

### 3.3 🔪 传统分割 — 覆盖实验 6、7、8、9

#### 3.3.1 边缘检测（实验 6）

| 算子 | 类型 | 特点 |
|------|------|------|
| **Sobel** | 一阶 | 结合平滑+差分，最常用 |
| **Canny** | 一阶优化 | 四步流程，双阈值，最优边缘 |
| **Roberts** | 一阶 2x2 | 简单快速，对噪声敏感 |
| **Prewitt** | 一阶 | 无中心权重 Sobel |
| **Laplacian** | 二阶 | 零交叉定位边缘中心 |
| **LoG** | 二阶+平滑 | 先高斯平滑再 Laplacian |

> **操作**：下拉选算子 → 点「执行边缘检测」

#### 3.3.2 阈值分割（实验 8）

| 方法 | 说明 |
|------|------|
| **Otsu** | 自动找最优阈值，最大化类间方差 |
| **迭代阈值** | 从均值开始迭代，收敛到稳定阈值 |

> **操作**：直接点按钮，结果实时显示在右画布

#### 3.3.3 区域生长（实验 9）⭐ 交互功能

```
操作步骤：
1. 在左侧「原图」画布上用鼠标点击一个位置（椎体区域）
   → 红点标记出现，状态栏显示种子点坐标
2. 调节「阈值」参数（默认20，越小越严格）
3. 点「区域生长」
   → 右画布显示生长出的区域
```

> **原理**：从种子点出发，BFS搜索邻域，灰度差≤阈值的像素被纳入

#### 3.3.4 K-means 聚类（实验 9）

| 选项 | 说明 |
|------|------|
| **纯灰度** | 只按像素灰度值聚类 |
| **灰度+空间** ☑ | 加上坐标信息，区域更紧凑 |

> **操作**：设 K 值(2-10) → 勾选空间模式 → 点「K-means 分割」

#### 3.3.5 霍夫直线检测

边缘提取 → 霍夫变换找直线 → 绿色线段叠加在原图上，用于**椎体终板检测**。

#### 3.3.6 纹理分析（实验 7）

| 方法 | 说明 |
|------|------|
| **GLCM** | 灰度共生矩阵，输出5种 Haralick 特征 |
| **LBP** | 局部二值模式，纹理局部特征图 |
| **Gabor** | 多方向多尺度纹理滤波响应 |

> 结果在状态栏显示数值（GLCM）或右画布显示特征图（LBP/Gabor）

### 3.4 🧠 深度学习 U-Net — 覆盖模块 4

#### 网络信息
- 架构：经典 2D U-Net（编码器-解码器 + 跳跃连接）
- 参数量：~500K（base_filters=32 轻量版）
- 输入：128×128 单通道灰度切片
- 输出：10 类（1背景 + 9节椎体）

#### 训练流程
```
1. 确认 CT 和标注数据已加载
2. 设置参数：
   - Epochs: 5-20（建议先 5 试跑）
   - Batch: 2（CPU 限制）
   - LR: 0.0001
3. 点「🚀 开始训练」
   → 控制台输出 loss 和 val_dice
   → 模型保存在 output/unet_vertebra.pth
```

> ⚠️ CPU 训练约 5-15 分钟/5 epochs，耐心等待。控制台会实时打印进度。

#### 推理预测
```
1. 训练完成或点「📂 加载预训练模型」
2. 点「🔮 推理预测」
   → 右画布显示 U-Net 分割结果
```

#### 后处理
点「**后处理**」→ 连通域分析去噪 + 沿脊柱方向编号椎体节段

#### 评估指标速查

| 指标 | 范围 | 含义 |
|------|------|------|
| **Dice** | [0,1] | 分割重叠度，1=完美 |
| **IoU** | [0,1] | 交并比 |
| **MSE** | [0,∞) | 均方误差，越小越好 |
| **MI** | [0,∞) | 互信息，越大越相似 |
| **NMI** | [0,1] | 归一化互信息 |

## 六、常见问题

**Q: 手动 FFT 误差较大？**
A: 非2的幂尺寸图像(61宽)会填充到2的幂做FFT，有轻微量化误差。频域滤波视觉效果不受影响。如需精确值，用 NumPy `np.fft.fft2` 对比。

**Q: 区域生长面积很小？**
A: 增大阈值（如 20→40），或在更亮的椎体区域点击种子点。

**Q: U-Net 训练太慢？**
A: 可将 Epochs 设为 3，image_size 设为 64。这是教学演示，实际分割主要用传统方法。

**Q: 想用自己的数据？**
A: 点「📁 文件导入 → 加载 CT 数据」，选你的 `.nii` 文件即可。
