## 第二章　实验原理

### 2.1 医学影像数据基础

**CT 值与窗宽窗位**：CT（Computed Tomography）图像中每个体素的像素值定义为 CT 值，单位为 Hounsfield Unit (HU)，其计算公式为：

$$\text{CT值} = \frac{\mu - \mu_{\text{水}}}{\mu_{\text{水}}} \times 1000$$

其中 $\mu$ 为组织的 X 射线衰减系数。水的 CT 值为 0 HU，空气为 -1000 HU，骨皮质可达 +1000~+3000 HU。

由于人眼只能分辨约 30 个灰度级，而 CT 值范围通常覆盖 -1000 到 +3000 HU（约 4000 个灰度级），远超人眼分辨能力，因此需要窗宽窗位（Window Width / Window Level）调节将感兴趣的 HU 范围线性映射到 0-255 显示灰度：

$$I_{\text{display}}(x,y) = \begin{cases} 0, & I(x,y) < \text{WL} - \text{WW}/2 \\ 255, & I(x,y) > \text{WL} + \text{WW}/2 \\ \frac{I(x,y) - (\text{WL} - \text{WW}/2)}{\text{WW}} \times 255, & \text{otherwise} \end{cases}$$

**MPR 多平面重建**：3D CT 体数据以三维数组存储，沿三个正交坐标轴提取切片：
- 轴状面（Axial）：垂直于身体长轴，即沿 Z 轴切片
- 冠状面（Coronal）：垂直于前后方向，即沿 Y 轴切片
- 矢状面（Sagittal）：垂直于左右方向，即沿 X 轴切片

### 2.2 灰度变换

**Gamma 变换**：$s = c \cdot r^{\gamma}$，当 $\gamma < 1$ 时扩展暗部，$\gamma > 1$ 时扩展亮部。

**对数变换**：$s = c \cdot \log(1 + r)$，压缩高灰度值，增强暗部细节。

**分段线性变换**：在不同灰度区间使用不同的线性映射，实现对比度拉伸或灰度切割。

### 2.3 直方图均衡化

通过将原始图像的灰度直方图变换为均匀分布来增强对比度。变换函数为累积分布函数（CDF）：

$$s_k = (L-1) \sum_{j=0}^{k} p_r(r_j)$$

其中 $p_r(r_j)$ 为灰度级 $r_j$ 的概率密度。

### 2.4 二维离散傅里叶变换与频域滤波

**2D DFT**：

$$F(u,v) = \sum_{x=0}^{M-1} \sum_{y=0}^{N-1} f(x,y) \cdot e^{-j2\pi(ux/M + vy/N)}$$

本实验手动实现了 Cooley-Tukey 基-2 递推 FFT 算法。频域滤波通过在频谱中心化后乘以滤波器函数 $H(u,v)$ 实现：

- **理想低通**：$H(u,v) = \begin{cases} 1, & D(u,v) \leq D_0 \\ 0, & D(u,v) > D_0 \end{cases}$
- **巴特沃斯低通**：$H(u,v) = \frac{1}{1 + [D(u,v)/D_0]^{2n}}$
- **高斯低通**：$H(u,v) = e^{-D^2(u,v)/(2D_0^2)}$

高通滤波器为 $H_{\text{HP}} = 1 - H_{\text{LP}}$。

### 2.5 边缘检测

**梯度算子（一阶）**：使用 Sobel 核计算梯度幅值 $M(x,y) = \sqrt{G_x^2 + G_y^2}$。

**Canny 检测器（四步流程）**：(1) 高斯平滑 → (2) 梯度幅值与方向计算 → (3) 非极大值抑制（NMS） → (4) 双阈值边缘连接（滞后阈值）。

**LoG（Laplacian of Gaussian）**：先高斯平滑再 Laplacian 二阶微分，通过零交叉检测定位边缘。

$$\nabla^2 G(x,y) = -\frac{1}{\pi\sigma^4} \left[1 - \frac{x^2 + y^2}{2\sigma^2}\right] e^{-\frac{x^2+y^2}{2\sigma^2}}$$

### 2.6 灰度共生矩阵（GLCM）

GLCM 统计图像中满足特定空间关系的像素对 $(i, j)$ 出现的频次。对方向 $\theta$ 和距离 $d$：

$$P(i,j|d,\theta) = \#\{(x_1,y_1),(x_2,y_2) | I(x_1,y_1)=i, I(x_2,y_2)=j, \overrightarrow{(x_1,y_1)(x_2,y_2)} \parallel \theta, |\overrightarrow{(x_1,y_1)(x_2,y_2)}| = d\}$$

本实验提取 5 种 Haralick 特征：对比度、能量、熵、逆差分矩（IDM）、自相关。

### 2.7 分割算法

**Otsu 最大类间方差法**：遍历灰度级 $t$，使类间方差 $\sigma_B^2(t) = \omega_0(t)\omega_1(t)[\mu_0(t) - \mu_1(t)]^2$ 最大。

**区域生长**：从种子点出发，BFS 搜索满足灰度差阈值的邻域像素，逐步扩展区域。

**K-means 聚类**：最小化类内平方和 $J = \sum_{i=1}^{k} \sum_{x \in C_i} \|x - \mu_i\|^2$，迭代分配-更新直至收敛。

### 2.8 图像配准与相似性度量

**互信息（MI）**：$I(A,B) = \sum_{a,b} p(a,b) \log_2 \frac{p(a,b)}{p(a)p(b)}$，衡量两幅图像共享的信息量。

**Dice 系数**：$\text{Dice} = \frac{2|A \cap B|}{|A| + |B|}$，最常用的分割重叠度评估指标。

### 2.9 U-Net 网络结构

U-Net 是一种经典的编码器-解码器架构，因形状对称如字母 "U" 而得名。其核心设计包括：

- **编码器（收缩路径）**：4 个下采样块（2×2 MaxPool + 双卷积），逐层提取高维语义特征
- **瓶颈层**：连接编码器与解码器的最深层
- **解码器（扩张路径）**：4 个上采样块（2× 双线性插值 + 跳跃连接拼接 + 双卷积），逐层恢复空间分辨率
- **跳跃连接（Skip Connection）**：编码器每层的特征图直接拼接到解码器对应层，保留空间细节信息


## 第五章　实验结果与分析

### 5.1 数据可视化

实验使用 `view_raw_data.py`（独立数据展示工具）对原始 CT 体数据和标注数据进行三平面可视化。矢状面（161×338 px）完整展示了脊柱纵向结构，9 节椎体在标注中以红色轮廓精确标出。矢状面方向的有效标注切片集中在第 33-113 层（共 81 层），每层标注 3-9 节椎体不等。

### 5.2 传统分割方法对比

在矢状面中间层（index=80）上对四种传统方法进行评估，以金标准标注为参考：

| 方法 | Dice ↑ | IoU ↑ | 说明 |
|------|--------|-------|------|
| Otsu 阈值 | 0.512 | 0.344 | 自动阈值，速度快但易受软组织干扰 |
| 迭代阈值 | 0.498 | 0.331 | 与 Otsu 接近，收敛于 T≈117.5 |
| 区域生长 | 0.423 | 0.268 | 依赖种子点选取，对阈值敏感 |
| K-means (k=3) | 0.610 | 0.439 | 灰度聚类效果最佳，骨骼类自动分离 |

**分析**：传统方法中 K-means 聚类表现最优（Dice=0.610），因为 CT 中骨组织（高 HU）、软组织（中 HU）和空气（低 HU）天然形成三个聚类。Otsu 次之（Dice=0.512），区域生长受种子点位置和阈值选择影响较大。

### 5.3 深度学习分割结果

**训练配置**：U-Net (base_filters=32), 81 张有效矢状面切片, image_size=256, batch_size=8, lr=1e-3, 20 epochs, GPU (RTX 3050)。

**训练曲线**：

| Epoch | Train Loss | Val Dice |
|-------|-----------|----------|
| 1 | 2.892 | 0.005 |
| 4 | 2.210 | 0.026 |
| 8 | 1.782 | 0.522 |
| 12 | 1.577 | **0.729** |
| 16 | 1.480 | 0.665 |

训练在 epoch 12 达到最佳验证 Dice=0.7294，后续出现轻微过拟合。总训练耗时 96 秒（约 5.6 秒/epoch）。

**与传统方法对比**：

| 方法 | Dice | 是否为自动 |
|------|------|----------|
| K-means (k=3) | 0.610 | ✅ 自动 |
| Otsu | 0.512 | ✅ 自动 |
| U-Net (ours) | **0.729** | ✅ 自动 |
| 区域生长 | 0.423 | ❌ 需人工选种 |

深度学习方法的 Dice 系数（0.729）较最佳传统方法 K-means（0.610）提升了约 19.5%，且完全自动，无需人工调整参数。

### 5.4 形态学后处理效果

对 U-Net 推理结果进行开运算（3×3 核）后接闭运算（5×5 核），可有效去除小面积噪点（去除约 3-8 个伪阳性区域）并填充椎体内部微小孔洞。

## 第六章　结论

本实验成功构建了一套覆盖医学图像处理课程全部 13 个实验核心技术的脊椎椎体逐节分割系统。主要成果包括：

1. **完整的技术覆盖**：手动实现 74 个 `my_` 前缀核心算法，涵盖图像读写、插值、灰度变换、直方图、FFT、边缘检测、纹理分析、阈值/区域/聚类分割、配准、评估、色彩空间、医学影像等方面。

2. **深度学习应用**：基于 PyTorch 实现 2D U-Net，在 NVIDIA RTX 3050 GPU 上 96 秒完成训练，验证 Dice 达到 0.7294，较传统最佳方法提升 19.5%。

3. **工程化系统**：基于 PyQt5 构建完整的图形化界面，集成 MPR 三视图、交互式种子点选取、参数实时调节、后台多线程计算、多方法自动评估等功能。

4. **AI 辅助开发实践**：首次将 Claude Code + DeepSeek-v4-pro 双模型协同开发工作流应用于医学图像处理算法开发，通过多代理并行协作在开发效率和代码质量方面取得了显著提升。

5. **标准化评估**：建立了统一的多方法对比评估框架，自动生成 Dice、IoU、MSE、MI、NMI 五项指标的对比报告。
