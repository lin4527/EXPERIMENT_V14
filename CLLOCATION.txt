三、系统模块详细设计
代码按以下 6 个模块分文件组织，每个模块独立封装，可单独运行也可通过 GUI 统一调用。
模块 1：医学影像数据读取与可视化模块
实现 NIfTI 格式 3D 体数据读取，提取像素间距、切片厚度等元信息
实现窗宽窗位调节算法，预设骨窗、软组织窗等临床预设
实现 MPR 多平面重建：支持轴状面、冠状面、矢状面正交切片提取，可通过参数切换切片位置
支持 jpg/png 等通用 2D 图像的读写与格式转换
模块 2：全算子图像预处理模块
所有算法提供「手动实现版本」与「库函数对照版本」，便于实验对比分析
灰度变换组：线性变换（支持斜率可调）、对数变换、Gamma 幂次变换、分段线性变换
直方图处理组：灰度直方图绘制、灰度统计特征计算、手动实现直方图均衡化
频域滤波组：二维 FFT/IFFT 手动实现、频谱中心化、理想 / 巴特沃斯 / 高斯低通 + 高通滤波器
采样插值组：空间分辨率重采样、最近邻插值、双线性插值手动实现、灰度级量化调整
彩色处理组：RGB↔HSI 空间转换、伪彩色映射、单通道滤波后通道融合
模块 3：传统方法分割基线模块
作为深度学习方法的对照组，核心算法全部手动编码
低层次特征提取：一阶 / 二阶边缘检测算子、霍夫直线检测（用于椎体终板检测）、傅里叶描绘子（分割轮廓形状描述）
纹理特征提取：四方向 GLCM 及 5 种纹理特征、LBP 特征图、Gabor 小波多尺度特征
阈值分割：迭代阈值法、Otsu 算法，输出椎体粗分割掩膜
区域与聚类分割：区域生长算法（支持种子点交互选取、灰度差阈值可调、八邻域生长）、K-means 聚类分割（支持仅灰度、灰度 + 空间坐标两种特征方案）
模块 4：深度学习椎体逐节分割核心模块
网络结构：实现经典 2D U-Net，输入 2D 切片，输出逐节椎体的多类别分割结果
数据处理：基于预处理模块构建数据增强流水线（灰度扰动、几何变换）
训练推理：包含数据集加载、训练循环、损失函数、推理全流程
后处理：结合连通域分析与椎体位置先验，对分割结果进行节段编号
模块 5：配准后处理与量化评估模块
配准校正：实现 2D 刚性配准、仿射配准，支持 MI/NMI 度量，将分割结果与模板配准校正
形态学后处理：腐蚀、膨胀、开运算、闭运算，去除噪点
量化评估：手动实现 Dice、IoU、MSE、MI、NMI 指标，自动生成「Otsu / 区域生长 / K-means / 深度学习」四种方法的指标对比表格
模块 6：GUI 交互集成系统
界面布局：左侧功能面板（文件导入、预处理、传统分割、深度学习、评估 5 组），右侧双画布显示区（原图 + 结果对照），底部参数调节滑块与指标输出区
